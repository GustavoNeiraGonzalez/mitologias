## ¿De que trata el proyecto?

Es una pagina que visualiza las historias mitologicas diferenciadas entre su propia mitologia, contiene login para el 
administrador (lo cual expira con el tiempo) quien a traves de jwt puede realizar las acciones de crear y modificar las historias 
además que los usuarios no logeados no verán ni podran acceder a los links de estas acciones
